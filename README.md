# basic_app
Trying github actions through a basic flask app

# Create environment
```
conda create -p venv python -y
```
# Activate env
```
conda activate venv/
```
# Create requirements.txt

```
pip install -r requirements.txt
```

# Git commands
```
git status
git add .
git commit -m <>
git push -u origin main
git remote -v
git log
git branch
```
